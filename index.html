CSS Property/Value Name Description
position: Position Property The default position of an element can be changed by setting its position property. The position property can take one of five values:
  • Static
  • Relative
  • Absolute
  • Fixed
  • Sticky
    •
Static is the default and does not need to be defined.
relative Relative Value Allows you to specify the position of an element relative to the offset properties: top, bottom, left, right.
absolute Absolute Value All other elements on the page ignore the element and act like it is not present on the page. Offset properties may be used alongside this property.
fixed Fixed Value Element can be fixed to a specific position on the page by accompanying with the offset properties above.
sticky Sticky Value Sticks element to a specified position and moves it along the page as the document is scrolled. Offset properties may be used alongside this property.
z-index Z-index Value Controls how far back or how far forward an element should appear on the web page when elements overlap, like depth. The higher the integer value used, the more forward the element will be.
display: Display Property Dictates if an element can share horizontal space with other elements. The display property can take one of three values:
  • inline
  • block
  • inline-block
    •
inline Inline Value Inline elements do not require a new line to display their content. They cannot be altered in size with the height or weight properties. This is the default display for some elements, such as <em>, <strong>, <a>.
block Block Value Fills the entire width of the page by default, they are not displayed in the same line as the content around them. Some default block-level elements are h1 through h6, <p>, and <div>.
inline-block Inline Block Value Combines features of both inline and block elements. Inline-block elements can appear next to each other and we can specify their dimensions using the width and height properties.
float: Float Property Moves an element as far left or as far right as possible. Often set using one of the values left or right.
clear: Clear Property Specifies how elements should behave when they bump into each other on the page. It can take on the left, right, both, or none values. These values indicate which side of the element will not any other element within the same containing element.
